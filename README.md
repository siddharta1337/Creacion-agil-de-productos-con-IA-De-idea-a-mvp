# Creación ágil de productos con IA: De idea a producto mínimo viable

Este es el repositorio del curso de LinkedIn Learning `Creación ágil de productos con IA: De idea a producto mínimo viable`. El curso completo está disponible en [LinkedIn Learning](https://www.linkedin.com/learning/instructors/carlos-solis).


Consulta el archivo Readme en la rama main para obtener instrucciones e información actualizadas.

Transforma tu proceso de creación de productos y acelera el paso de la idea al producto mínimo viable con el poder de la AI. En este curso descubrirás técnicas avanzadas para optimizar tus flujos y exprimir la agilidad al máximo. Aprende a mitigar los riesgos del vibe code, reduciendo la fricción y validando tus conceptos en tiempo récord. Convierte la inteligencia artificial en tu aliada estratégica de producción.

## Instrucciones

Encuentra los prompts y hoja de trabajo en las carpetas correspondientes

### Prompts
Todos los prompts del curso se encuentran disponibles en la carpeta `prompts` con el nombre correspondiente al video. 
Recuerda adaptarlos según las instrucciones de cada leccion.

### Hoja de Trabajo
En la carpeta `Hoja de trabajo`, encontrarás diferentes versiones y formatos del documento que utilizarás durante el curso.

Si quieres una version para imprimir (recomendado) utiliza el archivo formato PDF.

Si prefieres tomar notas directamente en la hoja de trabajo, puedes hacerlo utilizando el archivo Markdown.

Modifica y personaliza la hoja de trabajo según tus necesidades.
